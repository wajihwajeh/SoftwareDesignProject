# Run Project locally 
In the project root folder, you should run below command
```npm run dev```
Then project would run at 3000 port, you can access via `localhost:3000` in your web browser

# API Integrated Part
## Sign Up
URL: `http://localhost:3000/New-Login`
Integrated Backend API:  `http://localhost:5000/auth/register`

## Login
URL: `http://localhost:3000/Logon`
Integrated Backend API:  `http://localhost:5000/auth/login`

## Profile Create
URL: `http://localhost:3000/New-Profile`
Integrated Backend API:  `http://localhost:5000/profile`

## Fuel  Quote
URL: `http://localhost:3000/New-Galon`
Integrated Backend API: `http://localhost:5000/quote`
